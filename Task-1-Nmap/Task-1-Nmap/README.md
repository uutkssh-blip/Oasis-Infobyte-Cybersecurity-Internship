# Task 1 - Basic Network Scanning with Nmap

## Objective
To perform a network scan and identify open ports and services.

## Command Used
nmap -sV 192.168.1.10

## Open Ports Found
- 22 (SSH)
- 80 (HTTP)
- 443 (HTTPS)
- 3306 (MySQL)

## Result
The scan successfully identified active services running on the system.
